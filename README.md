# dsa-solutions

> Daily DSA problem solutions — generated and committed automatically by an AI agent.

![Solutions](https://img.shields.io/badge/Updated-Daily-brightgreen)
![Language](https://img.shields.io/badge/Language-Python-3776AB?logo=python&logoColor=white)
![AI](https://img.shields.io/badge/Powered_by-Llama_3-F55036)

One new problem solution every day, covering the full range of DSA topics with explanations, complexity analysis, and clean Python code.

## Topics covered

- Arrays & Hash Maps
- Trees & Graphs
- Dynamic Programming
- Sliding Window & Two Pointers
- Stack, Queue, Heap
- Binary Search
- Backtracking

## Structure

```
day-001/README.md   <- problem + approach + solution + complexity
day-002/README.md
...
```

Each `README.md` contains the problem statement, approach walkthrough, full Python solution, and time/space complexity breakdown.

## How it works

An AI agent runs daily via GitHub Actions, picks a problem, generates the full solution using Llama 3, and commits it here.

Source: [github-agent](https://github.com/Devgupta0/github-agent)
